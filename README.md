# epita-googlenews-seo

## TL;DR

```
    sudo install.sh
    scrapy crawl gnews
```

## Requirements

* python 2.7 or higher
* [scrapy](http://scrapy.org/)
* [feedparser](https://github.com/kurtmckee/feedparser)

## Coding

You need a python IDE like [PyCharm](https://www.jetbrains.com/pycharm/)


## Running

```scrapy crawl gnews```
